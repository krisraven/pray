# Commercial Licensing Guide

## Overview

The Pray CLI tool is available under a **Commercial Software License** that supports monetization while maintaining control over your intellectual property.

## License Types

### 1. Personal License (Free)
- **Cost:** Free
- **Users:** Individual personal use only
- **Distribution:** Not permitted
- **Modifications:** Not permitted
- **Redistribution:** Not permitted

**Best for:** Individual developers and enthusiasts

### 2. Commercial License (Paid)
- **Cost:** Variable (customizable per agreement)
- **Users:** Commercial/business use
- **Distribution:** Permitted (with license terms)
- **Modifications:** Contact for negotiation
- **Redistribution:** Permitted under license terms

**Best for:** Businesses, resellers, package managers

### 3. Enterprise License (Negotiated)
- **Cost:** Custom pricing
- **Users:** Large-scale deployment
- **Distribution:** Custom terms
- **Modifications:** Custom terms
- **Includes:** Priority support and custom features

## Sales Platforms & Strategy

### Option A: Individual Sales (Recommended for Starting)

**Platforms:**
1. **Gumroad** (Easiest setup)
   - Simple payment processing
   - Pay-what-you-want option
   - Link from your GitHub
   - ~95% payout to you

2. **Itch.io** (Community-friendly)
   - Developer-focused audience
   - Variable pricing support
   - Built-in updates

3. **Your Website** (Highest margins)
   - Full control
   - Requires Stripe/PayPal setup
   - Higher conversion potential

### Option B: Package Manager Distribution

**Premium Pricing Model:**
- Free version on GitHub (basic version)
- Paid versions on:
  - Homebrew (macOS)
  - Chocolatey (Windows)
  - Snap Store (Linux)
  - AUR (Arch Linux)

### Option C: Dual Licensing

**Hybrid approach:**
- **Free License:** MIT or GPL for open-source community
- **Commercial License:** Proprietary for businesses who want support/modifications
